Documentation:
--------------------------------
Activate the plugin and it will appear under the tools bar
Supported excel extensions are .csv , .xls and .xlsx only so make sure your excel file is in one of these format extensions
Select the status which the post will be acquire after successfully transfered ,

PUBLISH means the post will automaticlly get published and appear on the list of your posts on website post page.

DRAFT they will appear as drafts waiting to be published manually later

-------------------------------
Select the user to be atributed as the author of posts

-----------------------------------
To ensure you successfully post without errors, make sure in your excel sheet
The first column is for Post Title and
The second column is for Post Content.
Dont add any other column content except these to columns.
